AWS To-Do List

GitHub

Web application using Python's Flask library that will allow users to create a list of tasks they need to complete. It can be hosted using a AWS EC2 Istance IaaS, with a backend built upon AWS's NoSQL DynamoDB for its cost-effective flexabile storage. The Application was created with RESTful principles in mind, and makes use of an API Gateway with multiple HTTP endpoints to control traffic from the web application to Lambda functions that access the database. This helps extrapolate responsibility and spread it around as well as removing code that does not need to be running at all times in an EC2, converting them to a FaaS model.
